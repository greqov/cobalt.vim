# cobaltino.vim

One more Cobalt theme for vim. Yay!

## Preview

![cobaltino theme preview](./screenshot.png)

## Installation

With `vim-plug` manager:

```
Plug 'greqov/cobaltino.vim'
```

Or copy `cobaltino.vim` to `~/.vim/colors` directory.
